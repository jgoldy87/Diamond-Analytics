Diamond Analytics is a baseball analytics project built with Python that brings together live MLB data, interactive dashboards, and statistical analysis in one application.

This repository serves as the home for my MLB-focused projects, including tools that analyze current standings, league leaders, team performance, player statistics, and historical baseball data. The project uses the MLB Stats API to provide live information and is designed to expand with advanced analytics, interactive visualizations, and historical analysis using the Lahman Baseball Database.

Current Features
Live MLB division standings
Wild Card standings
League leaders
Daily scoreboard
Team Explorer
Interactive charts and visualizations with Plotly
Streamlit-based web application
